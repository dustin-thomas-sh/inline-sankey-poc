Inline Sankey Diagrams

Word-sized flow visualizations embedded directly in text — proof of concept

What Is This?

Edward Tufte coined the term sparkline for "small, intense, word-sized graphics." An inline sankey (or "sparksankey") applies that idea to flow data — showing how quantities split and merge across stages, right inside a sentence, without breaking reading flow.

Technical Notes

Each inline sankey is a <svg> with display: inline-block; vertical-align: middle. Layout is computed by d3-sankey. No labels at this scale — hover for detail. The entire page is a single HTML file with D3 loaded from CDN. Zero build steps.
